# 📔 Buku Catatan Pro
> **Catat ide jadi lebih gampang, kelola tugas jadi lebih ringan.**

Aplikasi ini dibuat buat kamu yang suka lupa naruh ide atau sering kelabakan nyari daftar tugas. Tampilannya bersih, simpel, dan nggak bikin pusing.

---

## 🔥 Apa saja yang bisa dilakukan?

*   **📝 Catat Apa Saja**: Mau nulis ide dadakan atau draft tulisan? Masukin aja ke sini. Tampilannya kotak-kotak (grid) biar enak dilihat.
*   **✅ Kelola Tugas**: Ada bagian khusus buat daftar tugas (to-do list). Tinggal liat, kerjain, beres!
*   **🔍 Cari Nggak Pakai Lama**: Lupa naruh catatan? Tinggal ketik judulnya di kolom cari, langsung ketemu.
*   **🎨 Editor yang Lengkap**: Pas lagi nulis, ada menu buat ganti warna, masukin foto, atau rekam suara.
*   **⚡ Ringan & Cepat**: Nggak pakai loading lama, langsung buka, langsung catat.

---

## 🛠️ "Jeroan" Aplikasi
Aplikasi ini dibangun pakai teknologi terbaru:
*   **Flutter**: Biar aplikasinya mulus di HP.
*   **Material 3**: Biar desainnya terlihat modern dan kekinian.
*   **Warna Emas**: Ikonnya pakai warna kuning emas biar terlihat elegan tapi tetap simpel.

---

## 🚀 Cara Pakai di Laptop Kamu
1.  **Download/Clone** dulu folder ini.
2.  Buka terminal/CMD, masuk ke folder aplikasinya.
3.  Ketik `flutter pub get` buat ambil bahan-bahannya.
4.  Terakhir, ketik `flutter run` buat nyalain aplikasinya di HP atau emulator.
